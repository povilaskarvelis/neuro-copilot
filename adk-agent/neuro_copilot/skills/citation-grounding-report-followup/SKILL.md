Use this skill when a follow-up question asks for exact paper IDs, cleaner references, source validation, or support behind a specific report claim.

Instructions:
1. Keep the lookup tied to the cited claim or reference in question.
2. Prefer one focused metadata or identifier recovery step over broad literature rediscovery.
3. If a citation is incomplete, recover the identifier and answer from the strongest available source.
4. When clarifying support for several named items, keep each citation attached to the exact dataset, paper, or claim it supports.
5. Do not expand into bibliography-building unless the user explicitly asks for it.
6. Load `references/citation-followup-playbook.md` before answering citation-heavy follow-ups.
