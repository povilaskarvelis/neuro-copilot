Use this reference for citation-focused report follow-up.

- Use `search_pubmed` or `search_pubmed_advanced` when the likely answer is a PMID-backed paper.
- Use `search_openalex_works` when DOI recovery or broader paper matching is easier there.
- Use `get_pubmed_abstract` after you have a PMID and need quick confirmation.
- Use `get_paper_fulltext` only when the follow-up explicitly needs details that abstract metadata does not contain.
- If several claims or datasets are mentioned together, answer with item-local citations rather than one paragraph-level citation bundle.
