Use this reference when executing a comparative step.

- Keep the same comparison dimensions across all entities in the step.
- If one side has only indirect or weaker evidence, say so explicitly.
- Prefer statements like "A leads on safety, B leads on efficacy" or "A is better supported, B is more speculative" instead of oversimplifying.
- If the comparison cannot be made fairly from the available evidence, mark the limitation rather than forcing a conclusion.
