Use this reference when executing GEO dataset searches.

- Use `search_geo_datasets` for discovery across disease, perturbation, tissue, organism, and accession clues.
- Use `get_geo_dataset` only after you have a likely accession or GEO UID.
- Treat GSE/GSM/GPL/GDS record types distinctly when they matter to the task.
- If the step needs actual biological interpretation, pair GEO retrieval with literature or another evidence source rather than treating GEO metadata alone as sufficient.
