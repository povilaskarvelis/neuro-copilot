Use this reference when the objective is comparative.

- Organize the plan around shared dimensions instead of reviewing entity A fully and entity B fully in isolation.
- Typical dimensions:
  efficacy or association strength, safety or risk, mechanistic plausibility, selectivity, tractability, biomarker context, trial maturity, and evidence quality.
- If one side has weaker or shallower evidence, make that a comparison finding rather than smoothing it over.
- Add a final synthesis step when the user asks which option is stronger, safer, more tractable, or better supported.
