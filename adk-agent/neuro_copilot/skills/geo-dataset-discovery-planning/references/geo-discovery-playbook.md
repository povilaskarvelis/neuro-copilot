Use this reference when GEO is the main dataset source.

- Use `search_geo_datasets` to discover candidate GSE, GSM, GPL, or GDS records from disease, tissue, perturbation, assay, or accession clues.
- Follow up with `get_geo_dataset` only for the records most likely to answer the question.
- Keep GEO discovery focused on study retrieval and metadata interpretation; if the user really wants biological conclusions, plan a separate corroboration step.
- When organism or study type is crucial, make that constraint explicit in the step goal and completion condition.
