Use this reference for variant-focused report follow-up.

- Use `search_variants_by_gene` when the report names a gene but not a specific rsID or HGVS string.
- Use `get_variant_annotations` for multi-source annotation context and `annotate_variants_vep` for consequence prediction detail.
- Use `search_civic_variants` or `search_civic_genes` for oncology-specific interpretation.
- Make conflicts and evidence level explicit instead of collapsing them into one verdict.
