name: variant-interpretation-report-followup
description: Use for report follow-up questions that need clarification of a specific variant, rsID, HGVS form, or competing interpretation.
---

Use this skill when a follow-up question is asking how to interpret a variant mentioned in or adjacent to the report.

Instructions:
1. Keep the lookup tied to the specific variant or small candidate set relevant to the follow-up.
2. If only the gene is known, find the likely variant first before using annotation tools that require explicit variant identifiers.
3. State clearly when the available evidence is gene-level, oncology-specific, predictive-only, or conflicting.
4. Do not broaden into exhaustive variant cataloging unless the user explicitly asks for it.
5. Load `references/variant-followup-playbook.md` before answering variant-heavy follow-ups.
