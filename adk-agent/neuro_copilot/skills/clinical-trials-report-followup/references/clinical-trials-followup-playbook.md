Use this reference for trial-focused report follow-up.

- Use `get_clinical_trial` when the report or question already points to a specific NCT.
- Use `search_clinical_trials` when the trial identifier is not yet known but the intervention or condition is.
- Use `get_dailymed_drug_label` or `search_fda_adverse_events` only when the follow-up is about warnings or post-market risk.
- Keep the answer tied to the follow-up scope instead of turning it into a fresh landscape analysis.
