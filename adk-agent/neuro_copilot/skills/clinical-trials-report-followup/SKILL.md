name: clinical-trials-report-followup
description: Use for report follow-up questions about specific trials, statuses, outcomes, labels, or safety context without reopening a full investigation.
---

Use this skill when the follow-up question is about a specific NCT, trial status, sponsor, phase, outcome, label warning, or post-market safety signal already adjacent to the report.

Instructions:
1. Keep the lookup scoped to the specific trial or small trial set relevant to the follow-up.
2. Prefer direct trial-record inspection over aggregate landscape tools when the user asks about a named or implied study.
3. Use label or FAERS context only when the follow-up is truly about safety or warnings.
4. Do not expand into a new therapeutic landscape review unless the user explicitly asks for one.
5. Load `references/clinical-trials-followup-playbook.md` before answering trial-heavy follow-ups.
