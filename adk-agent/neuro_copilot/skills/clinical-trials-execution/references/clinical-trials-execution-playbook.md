Use this reference when executing trial-centered steps.

- Use `search_clinical_trials` to discover candidate NCTs, then `get_clinical_trial` for direct record interpretation.
- Use `summarize_clinical_trials_landscape` for aggregate phase/status counts, not as a substitute for reading key trial records.
- If the question is about current US warning language, use `get_dailymed_drug_label`.
- If the question is about post-market signals rather than label text, use `search_fda_adverse_events`.
- Keep outcome measures, enrollment, sponsor, and status as separate fields in your reasoning.
