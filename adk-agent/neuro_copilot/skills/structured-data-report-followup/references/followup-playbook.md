Use this reference for report follow-up requests that depend on structured data.

- Stay within the specific claim, table, cohort, biomarker, gene, or endpoint the user asked about.
- Prefer quick confirmation of the report's existing evidence over broad discovery.
- Use schema inspection only if it directly unblocks a single focused lookup.
- If the answer requires multiple datasets, extensive reconciliation, or broader hypothesis generation, stop and defer to the full research workflow.
