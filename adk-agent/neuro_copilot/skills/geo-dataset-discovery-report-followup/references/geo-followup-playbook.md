Use this reference for GEO-related report follow-up.

- Use `get_geo_dataset` when the accession or GEO UID is known.
- Use `search_geo_datasets` when you need to recover the relevant GSE/GSM/GPL/GDS record.
- Keep organism, assay, tissue, perturbation, and accession type explicit in the answer when relevant.
- If the follow-up asks for biological interpretation rather than dataset metadata, answer within the report scope and avoid reopening the full investigation.
