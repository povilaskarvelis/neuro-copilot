description: Use for report follow-up questions that ask why the report leaned one way, how strong the evidence is, or how to interpret mixed findings.
---

Use this skill when the follow-up question is mainly about confidence, source strength, or why some evidence mattered more than other evidence in the report.

Instructions:
1. Explain the weighting logic directly: which source types are strongest here, which are weaker, and why.
2. Distinguish disagreement from uncertainty. Not all low-confidence findings are contradictory.
3. Be explicit when the report's conclusion rests on better source quality rather than greater source count.
4. Keep citations attached to the exact claim or comparison point being explained.
5. Load `references/evidence-weighting-followup.md` before answering evidence-strength follow-ups.
